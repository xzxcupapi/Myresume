I designed and developed the frontend interface for a travel website named TriPedia using React, Vite, and Tailwind. Ensuring both responsiveness and appealing aesthetics, the focus was on an intuitive user experience. I implemented dynamic React components and customized layouts using Tailwind for styling. The project successfully delivered an engaging and responsive visual experience for both desktop and mobile users.
